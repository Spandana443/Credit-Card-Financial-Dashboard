# Power BI Dashboard Project

## Overview
This project presents an interactive Power BI dashboard built to analyze key business metrics and support data-driven decision-making.

## Tools Used
- Power BI
- SQL
- Excel / CSV
- DAX
- Data Cleaning

## Dashboard Features
- KPI tracking
- Trend analysis
- Category-wise breakdown
- Interactive filters and slicers
- Business insights summary

## Project Workflow
- Collected and cleaned raw data
- Transformed data for reporting
- Built data model and relationships
- Created DAX measures
- Designed dashboard visuals and filters

## Key Insights
- Identified major trends and performance changes
- Highlighted top-performing categories / segments
- Supported business decisions through interactive reporting

## Files Included
- `dashboard.pbix` – Power BI dashboard file
- `images/` – dashboard screenshots
- `data/` – sample dataset if shareable

## Note
If the `.pbix` file does not open in browser preview, download it and open it in Power BI Desktop.
